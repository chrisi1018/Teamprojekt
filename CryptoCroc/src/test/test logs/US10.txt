US10 (GIT: c98288c)

1. Ich waehle das Caesar-Verfahren aus. Gebe einen Text in das "Klartextfeld" ein. 
Waehle eine Schluessel und druecke auf den Button "verschluesseln".
Erwartung:	Der Text im "Klartextfeld" wird, mit dem ausgewaehlten Schluessel, verschluesselt 
		und dieser verschluesselte Text, wird im "Geheimtextfeld" ausgegeben.
Realitaet:	Entspricht den Erwartungen.

2. Ich waehle das Caesar-Verfahren aus. Gebe einen Text in das "Geheimtextfeld" ein.
Waehle den Schluessel und druecke auf den Button "entschluesseln".
Erwartungen:	Der Text im "Geheimtextfeld" wird, mit dem ausgewaehlten Schluessel, entschluesselt
		und dieser entschluesselte Text, wird im "Klartextfeld" ausgegeben.
Realitaet:	Entspricht den Erwartungen.

3. Ich waehle das Monoalphabetische-Verfahren aus. Gebe einen Text in das "Klartextfeld" ein.
Waehle die Schluessel und druecke auf den Button "verschluesseln".
Erwartung:	Der Text im "Klartextfeld" wird, mit den ausgewaehlten Schluesseln, verschluesselt 
		und dieser verschluesselte Text, wird im "Geheimtextfeld" ausgegeben.
Realitaet:	Entspricht den Erwartungen.

4. Ich waehle das Monoalphabetische-Verfahren aus. Gebe einen Text in das "Geheimtextfeld" ein.
Waehle die Schluessel und druecke auf den Button "entschluesseln".
Erwartungen:	Der Text im "Geheimtextfeld" wird, mit den ausgewaehlten Schluesseln, entschluesselt
		und dieser entschluesselte Text, wird im "Klartextfeld" ausgegeben.
Realitaet:	Entspricht den Erwartungen.
